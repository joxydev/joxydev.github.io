# Голосовой переводчик — Demo

Учебный проект: простой цепочкой STT → Translate → TTS демонстрирующий работу с браузерными Web Speech API и публичным переводческим API.

Точка входа
- `index.html` — основной интерфейс демо (папка `voice-translator`).

Ключевой функционал
- Запись речи в браузере и распознавание (Web Speech API).
- Перевод распознанного текста через LibreTranslate (HTTP POST).
- Воспроизведение перевода голосом (Web SpeechSynthesis) с возможностью выбора голоса.
- Обработка ошибок (CORS/сеть) и подсказки для запуска локального сервера/прокси.

Технологии
- Vanilla HTML/CSS/JS
- Web Speech API (SpeechRecognition, SpeechSynthesis)
- LibreTranslate (публичный экземпляр)

Запуск локально

1. Из корня репозитория запустите:

```bash
npx live-server ./voice-translator --port=8001
```

2. Откройте в браузере `http://127.0.0.1:8001`.

Советы при проблемах с переводом (CORS)
- Если вы видите ошибку CORS при обращении к публичному экземпляру LibreTranslate, можно:
	- развернуть локальный экземпляр LibreTranslate;
	- или использовать простой CORS-прокси при разработке (например, `cors-anywhere`);
	- либо протестировать перевод через сервер-обёртку.
