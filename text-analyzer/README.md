# Text Analyzer — Demo

Короткий demo для портфолио: проверка орфографии, грамматики и пунктуации с возможностью применять исправления.

Как работает
- Клиент отправляет текст на публичный LanguageTool API (`https://api.languagetool.org/v2/check`) и получает список ошибок и предложений.
- Результат подсвечивается в превью; предложения отображаются в панели, их можно применять по одному или всем сразу.

Запуск и ограничения
- Откройте `text-analyzer/index.html` локально или через `npx live-server`.
- Публичный endpoint LanguageTool может быть недоступен из браузера из-за CORS. В этом случае запустите серверный proxy или вызовите LanguageTool с сервера.

Пример команды для локального сервера

```bash
npx live-server ./ -p 8000
```

Дальше
